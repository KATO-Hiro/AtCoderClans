## 回答のラベルをカスタマイズ

- [AtCoder Label Customizer](https://chromewebstore.google.com/detail/atcoder-label-customizer/jfgmfcnlmgolnfadehfomajomolbadhl) - 「提出結果」ページで、回答状況を表すラベルの文字と背景色を変更できる。
