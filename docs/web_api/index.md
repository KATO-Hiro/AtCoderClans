コンテストやユーザに関する各種統計情報を取得することができます。

!!! warning "注意"

    公開されているAPIにリクエストを送る場合は、一定の間隔を空け、回数が多くなり過ぎないようにすることを強く推奨いたします。

- [Kyopro-Ratings](https://github.com/algon-320/Kyopro-Ratings) ![GitHub Repo stars](https://img.shields.io/github/stars/algon-320/Kyopro-Ratings?style=plastic) - 複数のプログラミングコンテストサイトのレーティングを一度に取得できる。[Codeforces](https://codeforces.com/)にも対応。
    - [ブログに競プロのレーティングを表示するようにした(その2)](https://algon-320.hatenablog.com/entry/2019/04/06/011234) - 作者による紹介記事。
