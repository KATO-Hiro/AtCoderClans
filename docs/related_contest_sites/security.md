### ライブラリ

#### セキュリティ関連のユーティリティ

- [ptr-yudai/ptrlib](https://github.com/ptr-yudai/ptrlib) ![GitHub Repo stars](https://img.shields.io/github/stars/ptr-yudai/ptrlib?style=plastic) - CTF用のPythonライブラリ。暗号・ネットワーク・セキュリティ上の弱点などに関する複雑なプログラムを簡単に書くことができる。
