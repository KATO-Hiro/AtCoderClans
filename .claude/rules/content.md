# Content Rules

## Entry format

Each entry follows this pattern:

```markdown
- [owner/repo](URL) ![GitHub stars](https://img.shields.io/github/stars/owner/repo?style=plastic) - descriptions in Japanese.
```

## Content organization

- Items sorted alphabetically or by popularity; further grouped by language or purpose when counts are large.
- Ordering: by medium → purpose → subcategory → alphabetical/popularity.

## Content policy (see `docs/contributing/design.md`)

- Include when recommended by users or maintainer.
- Exclude if author requests removal or the item could harm users.
- **Author intent takes priority over all other considerations.**
